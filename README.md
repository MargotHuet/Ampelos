# 🍇 Ampelos

Ampelos est un outil qui utilise l'intelligence artificielle pour aider à l'analyse des feuilles de vigne. Il est capable de prédire si une feuille de vigne est en bonne santé ou atteinte d'une maladie.

Le but de ce projet est d'aider les petites structures viticoles - qui ne peuvent pas investir dans du matériel agricole moderne -  à identifier la vigne malade.



## 🛣️ Roadmap

- Analyse des données : data visualisation, séparation du jeu de données en sous-catégories (train, test, validation), data augmentation, optimisation des données d'entrainement. 
- Création du modèle : création d'un réseau de neurones convolutif, entrainement du modèle sur les données train test split 
- Evaluation de la fiabilité : sur les ensembles de données d'entrainement et de validation. 
- Test du modèle.
